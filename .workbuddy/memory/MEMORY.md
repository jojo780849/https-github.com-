# 项目记忆

## 当前项目：生存割草游戏 (Vampire Survivors 风格)

### 技术栈
- 纯 HTML5 Canvas + 原生 JavaScript
- 多个单文件原型并行推进（`demo_melee.html` / `demo_ranged_juice.html` / `demo_ranged.html` / `game.html`）
- 无需构建工具，双击即可运行

### 已实现功能（主线 V15 + 远程爽感实验线 V3 - 截至 2026-04-18）


#### game.html（远程射击版）
1. **核心机制**：WASD 键盘 + 虚拟摇杆触屏移动、自动索敌射击、敌人追踪AI、碰撞检测、难度递增
2. **深度 Boss 战系统 (V11)**：状态机(chase/warn_aoe/aoe_explode/warn_dash/dashing/stunned)、AOE预警、冲刺攻击、多阶段战斗(60%/30%)、狂暴模式、5种Boss
3. **顶视角射击 (V10)**：俯视角角色（白发+橙色外套+红围巾）、手持枪械+枪口火焰、360度自动旋转、紫色调地板
4. **10种敌人 (V10)**：蛋壳怪/瓢虫/愤怒蛋壳/金甲虫/幽灵蛋/蜘蛛虫/皇家蛋/火焰虫/泰坦蛋/虫后
5. **手机竖版布局 (V8)**：450x600画布、三段式布局、大号虚拟摇杆、全屏沉浸
6. **开局天赋系统**：4种稀有度、6随机池、3次重骰
7. **经验与升级系统**：经验宝石+磁铁拾取+随机技能池
8. **武器系统 (8种)**：基础4+特殊4(Lv5+)
9. **被动属性**：移速/拾取/暴击/生命回复

#### demo_ranged_juice.html（远程爽感实验线 - V3）
1. **定位**：远程射击爽感对照样机，用来验证可回灌到主线的反馈模块，不替代近战主线
2. **三把远程主武器**：弹射枪（站定精准爆发）/ 散弹枪（移动走位型）/ 激光笔（持续命中型）
3. **6 套 JUICE 系统**：慢动作、连杀里程碑奖励、大字幕与白闪、暴击数字强化、子弹与 Boss 冲刺拖尾、Web Audio SFX
4. **连杀奖励峰值**：脉冲清场、短时极速射击、清屏核爆、短暂无敌
5. **基础框架**：手机竖版布局、升级选择、Boss UI 与战斗流程均可独立试玩

#### demo_melee.html（近战版 - 当前主力开发线）
1. **核心战斗模型**：主近战+副远攻混合型（方案C），近战为唯一攻击动作自动挥砍，远程武器变被动技能槽自动触发，单摇杆操控

2. **三把主武器**（V15平衡）：
   - 匕首：45°/55px/CD10/伤12，贴脸加成×1.5，连击+10%/层(max+50%)，持刀移速+20%。定位=贴脸单体+高机动
   - 长剑：110°/85px/CD25/伤25。定位=均衡
   - 巨斧：150°/110px/CD65/伤50，击退8，移速-15%，10帧前摇（蓄力光圈）。定位=清波之王但笨重
3. **武器专属升级树 (V2)**：每武器6种专属升级，升级池2/3武器相关
4. **打击感系统 (V2)**：顿帧/击退/屏幕震动/击杀粒子/伤害数字物理弹跳
5. **Boss 战 (V4)**：炎魔领主，800HP+等级缩放，三阶段，四种攻击(AOE/冲刺/召唤/散射)
6. **浮动摇杆 (V3)**：手指任意位置按下即创建摇杆中心，永不丢失控制，非线性响应+死区+平滑
7. **无敌帧系统 (V12)**：统一 damagePlayer() 入口、0.75秒无敌帧、闪烁视觉、方向击退（力度分级）、碰撞伤害改单次触发
8. **UI 梳理 (V13)**：摇杆居中、跟随玩家弧形血条（绿→橙→红+低血闪烁）、顶部UI精简(Lv/击杀/时间)、伤害数字弹性缩放+描边发光
9. **回血道具 (V14)**：5%掉落率、回复15%最大HP、磁铁吸引、浮动脉冲动画、绿色治愈特效、drops架构支持扩展
10. **攻击范围扩大 (V15)**：三武器等比增大range和弧角，索敌距离+60匹配
