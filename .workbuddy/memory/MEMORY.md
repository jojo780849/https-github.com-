# 项目记忆

## 当前项目：生存割草游戏 (Vampire Survivors 风格)

### 技术栈
- 纯 HTML5 Canvas + 原生 JavaScript
- 单文件结构 (`game.html`)
- 无需构建工具，双击即可运行

### 已实现功能 (V10)
1. **核心机制**
   - WASD 键盘移动 + 虚拟摇杆触屏移动
   - 自动索敌与射击
   - 敌人追踪 AI
   - 碰撞检测
   - 难度随时间递增

2. **顶视角射击系统 (V10 新增)**
   - 🎯 俯视角角色绘制（白发+橙色外套+红围巾）
   - 🔫 手持枪械，射击时有枪口火焰特效
   - 🔄 角色自动旋转面向攻击方向（360度全向）
   - 🎨 紫色调场景地板（带裂缝纹理和阴影区域）

3. **新敌人系统 (V10 新增)**
   - 🥚 蛋壳怪：白色蛋壳、红眼、锯齿嘴
   - 🐞 瓢虫：红色身体、黑点、黄眼
   - 😠 愤怒蛋壳：红色裂纹、冲锋行为
   - ✨ 金甲虫：金色闪亮、快速冲刺
   - 👻 幽灵蛋：半透明、相位穿越
   - 🕷️ 蜘蛛虫：8条动态腿、俯冲攻击
   - 👑 皇家蛋：紫色、戴皇冠、召唤小兵
   - 🔥 火焰虫：火焰光环、远程攻击
   - 🗿 泰坦蛋：巨型、石头质感、高血量
   - 👸 虫后：光环增益、粉色身体

4. **手机竖版布局 (V8)**
   - 📱 竖版画布 450x600，专为手机设计
   - 🎮 三段式布局：顶部UI + 游戏区 + 底部摇杆
   - 🕹️ 固定底部大号虚拟摇杆 (140x140)
   - 📊 紧凑型顶部状态栏
   - 🔒 全屏沉浸体验，禁止滚动缩放

5. **开局天赋系统**
   - 每局开始前选择 1-3 个天赋
   - 6 个随机天赋池，可重骰 3 次
   - 4 种稀有度：普通/稀有/史诗/传说
   
   | 稀有度 | 天赋示例 |
   |--------|----------|
   | 普通 | 疾风(移速)、坚韧(HP)、磁力(拾取)、幸运儿(多选项)、大号弹药 |
   | 稀有 | 吸血本能、狂战士、玻璃大炮、决斗者、寻宝者、幻影(闪避) |
   | 史诗 | 开局武器、巨人化、迷你化、致命打击 |
   | 传说 | 一击必杀、凤凰涅槃(复活)、雷神之力、时间减速、多重射击、超级回复 |

3. **经验与升级系统**
   - 击杀掉落经验宝石
   - 磁铁拾取效果
   - 升级时从随机技能池中选择

4. **武器系统 (8种)**
   - 基础：弹射、环绕护盾、爆裂魔法、幽灵召唤
   - 特殊(Lv5+)：连锁闪电、死亡射线、黑洞、陨石雨

5. **Boss 系统**
   - 每 60 秒生成 Boss
   - 4 种 Boss 类型

6. **敌人种族系统 (14种)**
   - 随时间解锁不同敌人
   - 各有独特行为和外观

7. **被动属性**
   - 移速、拾取范围、暴击、生命回复
